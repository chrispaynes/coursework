# Ticket Storage

Ticket storage showcases a simple "todo-list-style" scheduling system. Users can create, delete, update and delete tickets.

## Usage

```
Open "index.html" in a browser.
```

## Features
* Bootstrap
* CRUD Operations
* Closures
* Data Validation
* Data Persistence
* DOM Manipulation
* Event Listening
* Functional Programming
* Immutability
* Polymorphism
* Prototypes
* Recursion
